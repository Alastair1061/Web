Primer examen

1.- Que es hypertexto?
Es un conjunto estructurado de textos, gráficos, etcetera, unidos entre si por enlaces y conexiones logicas.

2.- Que es hypermedia?
Es un conjunto estructurado de textos y graficos, asi como voz, audio, fotografias, graficos interactivos, videos, realidad virtual, entre otros, unidos entre si por enlaces y conexiones logicas.

3.- Que caracteristicas tiene un lenguaje de etiquetas?
Nos permite estructurar un documento mediante el uso de etiquetas
Puede ser editado con un sencillo editor de textos
Facilitan el intercambio de información entre distintos programas.
No permite el uso de funciones aritmeticas, el uso de variales, ni el uso de controladores de flujo.

4.- Cuales son las ventajas que introduce html5?
Incorpora nuevas caracteristicas semánticas, posibilidades multimedia, nuevas funciones para formularios, etc.

5.- Cual es la funcion del motor de renderizado de un navegador web?
Transformar los documentos HTML y otros recursos de una página web en una representación visual interactiva en el dispositivo del usuario

6.- Menciona al menos 2 tecnologias de software para el desarrollo de paginas web y explicalas brevemente:
JavaScript - Permite la creación de páginas dnámicas, con código que puede ejecutarse.
CSS - Establece reglas de representación de un documento en un medio o dispositivo.
